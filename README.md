# 🌍 Wanderlust – Airbnb Clone

A full-stack Airbnb-style web application built using **Node.js, Express, MongoDB, and EJS**, featuring authentication, listings, reviews, bookings, maps, and responsive UI.

🔗 🔗 Live Demo: https://airbnb-clone-vvuj.onrender.com/listings


---

## 🚀 Features

### 🔐 Authentication & Authorization
- User signup/login using Passport.js
- Secure sessions stored in MongoDB
- Protected routes for listings, reviews, bookings

### 🏡 Listings
- Create, edit, delete listings
- Upload images via Cloudinary
- Geolocation using Mapbox
- Category-based filtering

### ⭐ Reviews
- Add & delete reviews
- Ratings with star UI

### ❤️ Wishlist
- Save listings to user wishlist
- Stored in MongoDB

### 📅 Bookings
- Date-based booking system
- Total price calculation
- User bookings dashboard

### 🗺 Maps
- Interactive Mapbox maps
- Marker for each listing

### 📱 UI & UX
- Fully responsive (mobile-first)
- Mobile bottom navigation
- Lazy loading images
- Dark mode
- Skeleton loaders

### 🧪 Testing
- Jest + Supertest
- API route testing
- App/server separation for testability
